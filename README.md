# rcktsim
Simple C++ 2D Rocket Simulation. School Project.


## Dependencies
* SFML

## Installation

On Arch:
    ```sudo pacman -S sfml```

On Ubuntu:
    ```sudo apt-get install libsfml-dev```

Then: ```make help```
